# Grade Tracker 🎓

A simple app to track student grades and calculate averages.

## 🚀 Features
- Add subjects and grades
- Display grade list
- Calculate average automatically

## 🛠️ Technologies
- HTML
- CSS
- JavaScript

## ▶️ How to Run
Open `index.html` in your browser.

## 🎯 Purpose
Practice arrays, DOM manipulation, and calculations.
