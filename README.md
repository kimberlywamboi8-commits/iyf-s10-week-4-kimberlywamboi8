# Week 4 JavaScript Fundamentals

This project contains two JavaScript practice applications:

- An interactive Calculator
- A Student Grade Tracker

Both projects were created to practice JavaScript fundamentals such as variables, data types, functions, control flow, arrays, and objects.

## Projects Included

### 1. Calculator

The calculator performs basic arithmetic operations using clickable buttons.

#### Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Clear button
- Delete button
- Display screen for input and results

#### File

- `calculator.html`

### 2. Student Grade Tracker

The grade tracker allows users to enter student names and scores, then automatically calculates grades and shows class performance.

#### Features

- Add student names
- Add student scores
- Automatic grade calculation
- Show total number of students
- Show class average
- Show top student
- Show pass/fail status

#### File

- `grade-tracker.html`

## JavaScript Concepts Used

- Variables
- Data types
- Functions
- Conditional statements
- Arrays
- Objects
- DOM manipulation
- Event handling

## Grading System

- `80 - 100` = A
- `70 - 79` = B
- `60 - 69` = C
- `50 - 59` = D
- `0 - 49` = F

## How to Run

1. Open `calculator.html` in your browser to use the calculator.
2. Open `grade-tracker.html` in your browser to use the student grade tracker.
